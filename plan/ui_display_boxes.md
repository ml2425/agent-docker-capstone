## Key Display Box Features

- **Provenance Display:** Always shows Title, Authors, and either PubMed ID (for articles) OR Filename (for PDFs)
- **Edit Buttons:** Available for both Triplets and MCQs
- **Regenerate Button:** For MCQ - triggers LLM to try again with current LLM selection
- **Image Generation:** Separate display box with LLM selector (ChatGPT 4o or Gemini 2.5)
- **All boxes use clear visual separation** (cards/borders) for easy identification
